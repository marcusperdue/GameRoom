Open source console launcher
GameRoom

Your games. One room.

GameRoom is a minimalist desktop launcher for GameCube, Wii, PS2, Xbox, Nintendo DS, PS1, and PSP libraries. It does not include games, BIOS files, or emulators.

Before You Download

Use your own legally dumped games and console files. Some emulators need BIOS or firmware files from your own hardware. These early builds are unsigned, so macOS and Windows may show security prompts.

macOS Security Note

If macOS says GameRoom is damaged, move GameRoom.app to Applications, then run xattr -dr com.apple.quarantine /Applications/GameRoom.app in Terminal. This is temporary for early unsigned builds; future notarized builds will open normally.
